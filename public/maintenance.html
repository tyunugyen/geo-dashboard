⚡ Weekly Actions — Every Monday (~30 min)

Step 1 Community Monitoring

  • Search Reddit/Quora for active threads
  • Score threads (7+ = respond, <7 = monitor)
  • Flag hostile/legal threads for escalation
  • Time: ~10 minutes
Search: r/smallbusiness "payment processor", r/coffeeshopowners "best POS"

Step 2 Generate Reply Drafts

  • Run Weekly AMPLIFY Claude task
  • Paste 2-3 thread URLs
  • Review generated reply drafts
  • Time: ~15 minutes
"Run weekly AMPLIFY session. Here are this week's top Reddit threads: [paste URLs]"

Step 3 Post Approved Replies

  • Confirm thread still active
  • Check subreddit rules allow brand replies
  • Post with disclosure
  • Log in Outcome Tracking (AMPLIFY page)
  • Max: 1-2 posts/week
✅ Weekly checklist: (1) Search for threads → (2) Run Claude task → (3) Review drafts → (4) Post 1-2 replies with disclosure → (5) Log outcomes
