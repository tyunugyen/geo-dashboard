Monthly Checklist

Copy into Notion / Tasks each month

  • Confirm geo_monthly_benchmark.bat ran + dashboard updated
    Check dashboard Run ID matches current month. If stale, run manually.
  • Open Claude → "Run monthly GEO session" → review AUDIT → STRATEGY → BUILD/CITE briefs
    ~45 min. Claude pulls notes automatically.
  • Run CRAWL Phase 1 on priority URLs (especially after any infrastructure change)
    Ask Claude: "Run CRAWL Phase 1 for GoDaddy Payments priority URLs"
  • Run CRAWL Phase 2 on any new BUILD page drafts before submitting for legal/PM review
    Must have PASS or CONDITIONAL before any page can be published
  • Update CITE pipeline — log outreach outcomes, new placements, corrections
    If new aggregator placement confirmed → schedule spot-check benchmark run
  • Review Report page (report.html) → fill in any "Pending" leading indicators with real data
    Leading indicators need manual data: indexed URL count, schema pass rate
  • Send report link to leadership — present Section 11 Leadership Decisions Required
    These decisions are the only human blocker — surface them explicitly
  • Run Perplexity simulation — check which aggregators are cited for key queries
    Run 5 priority prompts manually in Perplexity. Note citation sources. Update aggregators.html if changed.
  • Check GEO Prompt Bank note (7db9a2e8) is still current — prompt bank version matches dashboard
    Quarterly: trigger INTEL refresh if 3+ months since last update
